# TruthKeep v11 Polished Release Notes

This build keeps the full v11 core and adds a polished non-technical user shell.

## Preserved core

- Mathematical memory core
- Truth governance
- Graph governance
- Correction / superseded / why-not
- Invariants
- MCP stdio backend
- OpenClaw adapter
- Tests, specs, benchmarks, reports

## New shell

- No-code launcher with tkinter fallback
- Easy OpenClaw manifest by default
- Advanced OpenClaw manifest separated
- Windows/macOS/Linux launcher scripts
- No-code guide and OpenClaw setup guide

## Security posture

- Local-only
- MCP stdio only
- No HTTP daemon
- No open port
- No LAN/cloud by default
