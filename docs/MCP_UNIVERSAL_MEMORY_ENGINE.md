# TruthKeep v11 MCP Universal Memory Engine Upgrade

## Mục tiêu

TruthKeep không cố làm cloud, không mở HTTP daemon, không copy hướng nhớ rộng. Bản nâng cấp này tập trung vào một việc: biến TruthKeep thành **local MCP memory engine** có thể cắm vào nhiều AI agent/host nhất có thể.

```text
AI host / agent
  ↓ MCP stdio
TruthKeep MCP
  ↓ SQLite local DB
nhớ đúng + nhớ lâu + correction + superseded + why-not + graph sự thật
```

## Nguyên tắc không đổi

- Không đổi lõi memory/toán/graph cũ.
- Không thêm nhớ rộng/spreading activation.
- Không mở port.
- Không HTTP daemon.
- Không cloud mặc định.
- Easy Mode chỉ phơi 5 tool memory chính.

## 5 tool mặc định

| Tool | Mục đích |
|---|---|
| `memory_remember` | Ghi nhớ thông tin quan trọng/current truth. |
| `memory_recall` | Gọi lại ký ức đáng tin. |
| `memory_correct` | Sửa ký ức sai/cũ và giữ lịch sử why-not. |
| `memory_status` | Kiểm tra trạng thái hệ nhớ. |
| `memory_profile` | Xem profile/sở thích/dự án đã nhớ. |

## Lệnh mới

```bash
truthkeep mcp list-hosts
truthkeep mcp config <host>
truthkeep mcp install <host>
truthkeep mcp doctor <host>
truthkeep mcp smoke-test <host>
truthkeep mcp export-matrix
```

Host profiles hiện có:

```text
openclaw, claude, cursor, cline, roo, continue, vscode, generic
```

## Quy trình người dùng

### Người dùng phổ thông

```bash
truthkeep easy install
truthkeep mcp install openclaw
truthkeep mcp doctor openclaw
```

Sau đó restart OpenClaw.

### Dev dùng nhiều host

```bash
truthkeep mcp config claude
truthkeep mcp install cursor
truthkeep mcp doctor cursor
truthkeep mcp smoke-test generic
```

## Lý do phải có Universal MCP layer

MCP giúp một server local dùng chung cho nhiều host, nhưng mỗi host có đường dẫn config, cơ chế reload và mức hỗ trợ khác nhau. Vì vậy TruthKeep không chỉ cần `truthkeep-mcp`; nó cần profile, doctor, smoke-test, structured error, và compatibility matrix.

## Bảo mật MCP

Easy Mode theo deny-by-default:

- Không tool shell.
- Không tool xóa/export nguy hiểm mặc định.
- Không mở port.
- Không HTTP daemon.
- Không cloud.
- Chỉ 5 tool memory dễ hiểu.

Advanced tools nằm trong manifest riêng.

## Tiêu chí pass

- `truthkeep mcp list-hosts` chạy được.
- `truthkeep mcp config generic` sinh JSON hợp lệ.
- `truthkeep mcp install generic` ghi config local có backup nếu cần.
- `truthkeep mcp doctor generic` kiểm được startup-probe.
- `truthkeep mcp smoke-test generic` không treo.
- `truthkeep mcp export-matrix` tạo bảng compatibility.
- `python -m compileall` pass.
- Focused tests pass.

## Runtime Profile Defaults

MCP host configs generated by TruthKeep now include:

```json
"env": {
  "TRUTHKEEP_MODE": "easy",
  "TRUTHKEEP_TRANSPORT": "stdio",
  "TRUTHKEEP_OPEN_PORTS": "0",
  "TK_RUNTIME_PROFILE": "local"
}
```

`local` is the safest default for normal MCP usage because it keeps TruthKeep's correctness DNA enabled while keeping heavy research simulators out of agent hot paths.

Use:

```bash
truthkeep profile doctor
```

to verify the active profile.
