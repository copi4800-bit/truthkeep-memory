# OpenClaw Easy Setup

Run:

```bash
truthkeep easy install
truthkeep openclaw doctor
```

Then restart OpenClaw.

TruthKeep installs an Easy Mode manifest with only five default tools. The full advanced manifest is available as `openclaw.plugin.advanced.json` for developers.

## Manual MCP config

```bash
truthkeep openclaw config
```


## Public setup command

OpenClaw and npm/package users can also run:

```bash
truthkeep-setup
```

It launches the same Easy Mode setup.
