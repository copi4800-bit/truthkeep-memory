# Phase 131: Single Command Center UX Shell

## Goal
Give TruthKeep one dominant UX destination that unifies current truth, workflow loop, transition history, ordinary mode, operator mode, and deep inspection in a single command-center shell.

## Scope
- Add a command center shell surface and HTML report.
- Expose it through app, MCP, tool registry, and OpenClaw-facing metadata.
- Add demo/render scripts and regression tests.

## Non-Goals
- No retrieval, governance, or truth-math changes.
- No new storage schema.
- No hosted/web frontend.
