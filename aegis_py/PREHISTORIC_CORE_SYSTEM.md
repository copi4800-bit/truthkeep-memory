# TruthKeep Prehistoric Core System

Internal canonical design language for TruthKeep Memory.

## Governing Rule

Every beast must resolve through this chain:

1. biology
2. mathematics
3. architecture

The prehistoric system is valid only when all three layers are present.

## Runtime Compression Rule

All 23 beasts remain part of the conceptual core.

They do **not** become 23 public runtime modules.

Runtime stays compressed into:

- `memory`
- `retrieval`
- `hygiene`
- `profiles`
- `storage`
- `integration`
