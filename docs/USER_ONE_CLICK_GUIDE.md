# TruthKeep One-Click User Guide

## Windows

1. Unzip TruthKeep.
2. Double-click `installers/INSTALL_TRUTHKEEP_WINDOWS.cmd`.
3. Wait until you see `[OK] TruthKeep is installed and OpenClaw Easy Mode is ready.`
4. Restart OpenClaw.
5. Use `memory_status` to check it.

## macOS / Linux

1. Unzip TruthKeep.
2. Open Terminal in the folder.
3. Run:

```bash
./installers/INSTALL_TRUTHKEEP_MAC_LINUX.sh
```

4. Restart OpenClaw.
5. Use `memory_status`.

## What you should see in OpenClaw

Only five TruthKeep tools:

- `memory_remember`
- `memory_recall`
- `memory_correct`
- `memory_status`
- `memory_profile`

If you see dozens of advanced tools, you installed the advanced manifest by mistake.
